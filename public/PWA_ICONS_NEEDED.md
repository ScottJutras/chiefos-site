# PWA Icons Required

The manifest references these icon files. They must be added to make the PWA installable with the ChiefOS icon:

- `icon-192.png` — 192×192px, square, transparent or dark background
- `icon-512.png` — 512×512px, square, transparent or dark background

## Design guidance

- Background: `#0f1117` (matches app theme_color)
- Use the ChiefOS wordmark or "C" logomark in white
- "Maskable" safe zone: icon content should stay within the center 80% of the image

## Quick path

If you have the logo as SVG, use squoosh.app or figma export to create 192×192 and 512×512 PNG exports.
